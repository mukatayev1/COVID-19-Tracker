# COVID-19-Tracker-App
App that monitors latest statistic of the COVID-19 spread around the world and any country of the world.

## Description
App monitors the statistics of Coronavirus spread around the world using COVID-19 data API.
Also app allows users to choose any country and see chosen country's COVID-19 stats.

### See the video: [YouTube Link](https://youtu.be/igq8UU6Ekdo)

## Features
* World's total statistics including total numbers and new cases for specific country
* Search for country through list of countries and see chosen country's COVID-19 statistics
* Quickly search for countries by typing the name of the country into the search field
* System Dark Mode

* SnapKit
* Charts
* Search Controller
* 100% Programmatic UI

## Setup
To run this project, install necessary pods using Terminal:

Set the cd to the project folder. 

Install pods by running this command in Terminal
```
pod install
```
